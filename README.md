# ❤️ Heart Disease Prediction using Machine Learning

## 📌 Project Overview

This project focuses on predicting the presence of heart disease using multiple machine learning algorithms. By combining several datasets and applying advanced preprocessing techniques, the goal is to build a robust and accurate predictive model.

Heart disease is one of the leading causes of death worldwide. Early prediction can help in timely diagnosis and treatment, potentially saving lives.

---

## 🎯 Objectives

* To analyze clinical datasets related to heart disease
* To combine multiple datasets for improved performance
* To apply and compare different machine learning models
* To identify the most accurate model for prediction

---

## 📊 Datasets Used

This project uses multiple heart disease datasets combined into a single dataset to improve model performance and generalization.

Common features include:

* Age
* Sex
* Chest pain type
* Resting blood pressure
* Cholesterol level
* Maximum heart rate achieved
* Exercise-induced angina
* ST depression
* Number of major vessels
* Thalassemia

## 🔍 Methodology

### 1. Data Collection

* Multiple datasets were gathered and merged

### 2. Data Preprocessing

* Handling missing values
* Feature scaling
* Encoding categorical variables

### 3. Exploratory Data Analysis (EDA)

* Visualization of feature distributions
* Correlation analysis

### 4. Model Building

The following models were implemented:

* Logistic Regression
* Decision Tree
* Random Forest
* XGBoost
* LightGBM
* CatBoost

### 5. Model Evaluation

* Accuracy
* Precision
* Recall
* F1-score

---

## 📈 Results

* Ensemble and boosting models performed better than traditional models
* Combining multiple datasets improved overall performance and stability
* Random Forest, XGBoost, LightGBM, and CatBoost showed the best results
